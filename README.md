# BS-api

### 前言
---
###### BS项目是一个网上购书系统，采用最新的技术栈实现
### 项目介绍
---
###### BS项目是基于B/S架构的购书系统，使用Springboot+MyBatis实现。主要功能包括：图书浏览、购物车管理、订单管理、会员管理等模块
